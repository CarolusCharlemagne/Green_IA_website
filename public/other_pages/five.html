Conseils et informations

Dates et informations de collecte :
  • Rendez-vous en page 1 de l'application (celle la plus à gauche).
  • Dans le champ sous la carte, entrez votre code postal.
  • Cliquez sur le bouton 'Valider'
  • Toutes les informations sur les jours, ainsi que les types de déchets récupérés vous serons affichés (si disponibles dans votre ville).
Dates et lieux de dépots :
  • En page 2, vous avez la possibilité d'afficher les endroits où il vous sera possible de déposer vos déchets.
  • Sélectionnez les catégories de déchets dont vous voulez vous débarasser, parmis celles proposées.
  • Tous les lieux de récupérations à moins de 10 km de votre position vous seront affichés.
  • Pensez bien à vérifier les horaires ainsi que les catégories de déchets traités par les lieux que vous souhaitez visiter.
Scanner :
  • En page centrale, il vous est possible de scanner les produits alimentaires que vous consommez.
  • Chacun des produits scannés sera sauvegardé dans un fichier, que vous pouvez télécharger en cliquant sur le bouton dédié.
  • Il vous est également possible de vider cette liste .
